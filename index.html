Selected Projects

Multi-modal Fake News Detection

Our research involves examining news stories published on social media and news agencies, and identifying fake news by analyzing their image and text, as well as by employing machine learning algorithms like for embedding image and text, concatenation vectors and learning specific and corresponding features of text and image.
Code More

Transfer Learning: We explored several approaches for embedding text and image, and found that conceptual and semantic embedding worked best, Huggingface’s BERT model for text and ViT model for image. In the news agencies text, Big Bird model was more relevant and accurate due to the length of the documents.
Multi-modal Learning: We evaluated different algorithms to fuse text and picture vectors, and found that the Constrative Loss function and concatenation worked better for short texts, and attention based fusion for lengthy texts.

Automatic text classification

Actions and models that may be applied to various sorts of text for improved classification have been implemented in this project. It provides the embedding and classification model to apply to the text using a command. The basic analysis of the data is presented first and the analysis of the results at the end of training. Optuna adjusts the different parameters after picking the model to get the optimum outcome.

Code More

EDA: This phase shows experimental operations like text length, label distribution, and features visualization.
Augmentation: Different types of augmentation methods of words, documents and features are ready to use in this phase.
Embedding: In this phase, text embedding algorithms like Fasttext, Word2vec, and contextual embedding can be used to extract features.
Algorithms: Machine learning, deep learning, and transformer learning models are implemented to select and use in training. Parameter tuning is done on algorithm parameters to achieve the best result.

Automatic image classification

In this project, activities such as adding data, classifying images, assessing and visualising are conducted to categorise the input images. Simply place the input in the data folder and enter the options, such as what kind of data augmentation and classification model to use, in the settings file, then run it and pick the output.

Code More

Augmentation: Several techniques for adding data at the image level and extraction features level have been implemented in this phase.
Algorithms: Image classification models using deep and deeper learning models such as VGG, ResNet, etc can be selected and used in this phase.
Error Analysis: A comprehensive review of the final model, as well as the incorrectly identified photos, is provided in this step.
